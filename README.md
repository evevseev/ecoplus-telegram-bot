# ecoplus-telegram-bot

Internal Telegram admin bot for managing an electric-scooter rental fleet (Busy-Fly platform): look up a scooter by number, view status / battery / GPS / last order, send hardware commands (beep, tech mode, RGB, unlock battery), check daily earnings and active rides by region, and reconcile CloudPayments transactions.

> Internal ops tool for a specific business — not a public or reusable project.
